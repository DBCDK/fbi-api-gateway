# Related Subjects

Get subjects related to the result found. This query takes an array og string and return another array of strings with subjects related to input.

<InlineGraphiQL
  query={`query Example_RelatedSubjects ($q: [String!]!, $limit: Int) {
    recommendations {
      subjects(q: $q, limit: $limit) {
        traceId
        subject
      }
    }
}
`}
  variables={{
    q: ["harry", "magi"],
    limit: 10,
  }}
/>
