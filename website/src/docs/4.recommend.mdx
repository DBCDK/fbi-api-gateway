# Recommend

Get work or manifestation recommendations based on a pid.

Results returned as workIds (works)

<InlineGraphiQL
  query={`query ($pid: String!, $limit: Int!) {
  recommend(pid: $pid, limit: $limit) {
    result {
      work {
        workId
        titles {
          main
        }
        creators {
          display
        }
      }
    }
  }
}`}
  variables={{
    pid: "870970-basis:55139784",
    limit: 10,
  }}
/>

Results returned as pids (manifestations):

<InlineGraphiQL
  query={`query ($pid: String!, $limit: Int!) {
  recommend(pid: $pid, limit: $limit) {
    result {
      manifestation {
        pid
        titles {
          main
        }
      }
    }
  }
}`}
  variables={{
  "pid": "870970-basis:28995946",
  "limit": 10
}}
/>
