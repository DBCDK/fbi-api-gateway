# Recommend

Get work or manifestation recommendations based on a pid.

Results returned as workIds (works)

<InlineGraphiQL
  query={`query ($pid: String!, $limit: Int!) {
  recommend(pid: $pid, limit: $limit) {
    result {
      work {
        workId
        titles {
          main
        }
        creators {
          display
        }
      }
    }
  }
}`}
  variables={{
    pid: "870970-basis:55139784",
    limit: 10,
  }}
/>

Results returned as pids (manifestations):

<InlineGraphiQL
  query={`query ($pid: String!, $limit: Int!) {
  recommend(pid: $pid, limit: $limit) {
    result {
      manifestation {
        pid
        titles {
          main
        }
      }
    }
  }
}`}
  variables={{
    pid: "870970-basis:28995946",
    limit: 10,
  }}
/>

## Local Recommendations

When using 'localRecommend', results are limited to the agency associated with your access token.
The branchId parameter is used to further limit recommendations to a specific branch within that agency.

This example assumes that you have an access token for Aarhus and uses 775147 (Hasle Bibliotek) as branchId.

<InlineGraphiQL
  query={`query ($pid: String!, $limit: Int!, $branchId: String!) {
  localRecommend(pid: $pid, limit: $limit, branchId: $branchId) {
    result {
      work {
        workId
        titles {
          main
        }
        creators {
          display
        }
      }
    }
  }
}`}
  variables={{
    pid: "870970-basis:28995946",
    limit: 10,
    branchId: "775147",
  }}
/>
