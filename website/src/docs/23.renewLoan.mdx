# BIBDK Renew loan

Renew a loan through openuserstatus service.

It is possible to renew a loan at any agency the user is registered at.

⚠️ If the _dryRun_ argument is set to true. The actual service will never get called, but will return a deleted "true".

<InlineGraphiQL
  query={`mutation renewLoan($loanId: String!, $agencyId: String!, $dryRun: Boolean) {
  renewLoan(loanId: $loanId, agencyId: $agencyId, dryRun: $dryRun) {
    renewed
    error
    dueDate
  }
}`}
