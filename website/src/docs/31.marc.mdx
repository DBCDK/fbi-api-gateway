# MARC

Field for presenting bibliographic records in MARC format.

## Get by recordId

Gets the MARC record collection for the given record identifier, containing either standalone or head and/or section and volume records.

The record identifier `recordId` can be extracted from either a work `Work.marc.id` or an manifestation `Manifestation.marc.id`

The serialization format of the MARC record content, is _marcXchange_.

<InlineGraphiQL
  query={`
    query Example_GetMarcByRecordId($id: String!) {
      marc {
        getMarcByRecordId(recordId: $id) {
          id
          agencyId
          bibliographicRecordId
          contentSerializationFormat
          deleted
          content
        }
      }
    }
`}
  variables={{
    id: "870970:138824578",
  }}
/>

## Get by Manifestation

Access to a marc record can also be done by the `marc` field directly on a manifestaion

<InlineGraphiQL
  query={`
    query Example_MarcRecordManifestation($pid: String!) {
      manifestation(pid: $pid) {
        pid
        marc {
          id
          agencyId
          bibliographicRecordId
          contentSerializationFormat
          deleted
          content
        }
      }
    }
`}
  variables={{
    pid: "870970-basis:54829574",
  }}
/>

## Get by Work

Like manifestations, access to a marc record can also be done by the `marc` field on a work

<InlineGraphiQL
  query={`
    query Example_MarcRecordWork($id: String!) {
      work(id: $id) {
        workId
        marc {
          id
          agencyId
          bibliographicRecordId
          contentSerializationFormat
          deleted
          content
        }
      }
    }
`}
  variables={{
    id: "work-of:870970-basis:48953786",
  }}
/>
