# Inspiration

The following examples show how to fetch data used for inspiration.

## Categories

The categories field is used to retrieve works based on a given category.
Categories contains subcategories, which can hold different subcategory titles,
like materialTypes, platforms or _popular_ and _newest_ materials.

The result of a category can be either a _Work_ or a _Manifestation_.

<InlineGraphiQL
  query={`query ($limit: Int!) {
  inspiration {
    categories {
      games {
        title
        result(limit: $limit) {
          work {
            workId
            titles {
              main
            }
          }
        }
      }
    }
  }
}`}
  variables={{ limit: 10 }}
/>

Subcategories can be specified, by passing a _filters_ argument to the category. The filters argument can hold multiple subcategory titles,
like _nyeste_ and/or _populære_.

<InlineGraphiQL
  query={`query ($limit: Int!, $filters: [String!]) {
  inspiration {
    categories {
      games(filters: $filters) {
        title
        result(limit: $limit) {
          manifestation {
            pid
          }
        }
      }
    }
  }
}`}
  variables={{ limit: 1, filters: ["nyeste", "populære"] }}
/>
